# Shopping Cart

A mock store built for The Odin Project. It uses the CheapShark API to search and display real-time video game deals.

**[View Live Demo]()**

## Tech Stack

-   **Core:** React, Vite
-   **Styling:** CSS Modules
-   **Testing:** Vitest, React Testing Library

## Quick Start

```bash
# 1. Clone
git clone [https://github.com/Qu1dy/shopping-cart.git](https://github.com/Qu1dy/shopping-cart.git)

# 2. Install
cd odin-shopping-cart
npm install

# 3. Run
npm run dev

Features

    Search games via CheapShark API

    Add/Remove items from cart

    Client-side routing
```
